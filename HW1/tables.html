TABLES

This page is for displaying a table that we can use.

The making of a table

Making a table in HTML consists of using four different tags; <table>,<th>,<tr>, and <td>.

<table>
- Used to define the table
<th>
- Used to define the table header
<tr>
- Used to define each table row
<td>
- Used to define each table piece of data from the table

Example of a table

Class Name Room Time
CS 160 100 MW 8:00-9:50
CS 270 130 TR 1:00-2:50
MTH 340 80 MTWR 12:00-12:50
