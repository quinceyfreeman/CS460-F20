LISTS

This page is for displaying the different types of lists that we can use.

Ordered Lists

This is an ordered list. You can make an ordered list in HTML by using the <ol> element.
  1. One
  2. Two
  3. Three
  4. Four
  5. Five

Unordered Lists

This is an unordered list. This list can be make almost identically with the only difference being that the unordered element is written as <ul> element.
  • Four?
  • Maybe One
  • Possibly three
  • Who knows two
  • All thats left is five

Description Lists

And the last list that HTML has to offer is the description list. This list allows you to decribe items in your list using indentations, but for some reason decided not to so I used some colors to show the <dd> element. The description list can be made using the <dl> element.
